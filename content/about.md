Welcome to my blog. I am Dongjae Lee, a Ph.D. student in the School of Computing at KAIST and a member of [Prosys Lab](https://prosys.kaist.ac.kr/), advised by [Kihong Heo](https://kihongheo.kaist.ac.kr/).
My goal is to build trustworthy programming systems by combining formal methods and agentic AI.
Specifically, my research interests are as follows:
- **Proof-Carrying Spec-Driven Development:** Currently, coding assistants (Cursor, Claude Code, Antigravity, ...) require continuous human supervision. I aim to minimize this supervision by enabling coding assistants to autonomously verify their own code through formal verification.
- **Mixing Natural Language and Formal Language:** Currently, most vibe coding prompts are written in natural language, and even the development plans generated by LLMs are written in natural language. However, natural language is fundamentally unamenable to formal methods (static analysis, type checking, etc.), meaning there is no reliable way to verify correctness beyond manual human inspection or having the LLM review its own plan. I aim to develop a next-generation programming language that blends formal and natural languages at an appropriate level, enabling formal analysis while remaining as easy to write as natural language.
- **Security for Agentic Software:** Modern software increasingly incorporates agentic AI, where AI systems autonomously use natural language prompts and simple tool programs (such as Claude Code's SKILL feature). This behavior introduces novel security vulnerabilities that traditional program analysis cannot detect. Addressing these vulnerabilities requires new methods for analyzing program behavior.
